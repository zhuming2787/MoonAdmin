# 三、权限与安全

## 待完成

### 1、RBAC 权限模型：基于角色的权限控制（超级管理员 / 内容管理员 / 普通作者）

### 2、菜单权限：不同角色看到不同的侧边栏菜单

### 3、按钮权限：如普通作者看不到「删除他人文章」按钮

### 4、登录日志：记录用户登录时间 / IP / 设备，异常登录预警

### 5、操作日志：记录所有增删改操作，支持按用户 / 时间 / 操作类型查询

### 6、密码强度校验：设置密码复杂度规则（如长度 ≥8、包含大小写）

### 7、二次验证：删除 / 批量操作等高危操作需再次确认

### 8、IP 白名单：限制仅指定 IP 段可访问管理端

### 9、会话超时：无操作 30 分钟后自动登出

### 10、多端登录互斥：同一账号仅允许在一个设备登录

### 11、接口防刷：对高频请求接口添加限流策略

### 12、数据加密：敏感数据（如密码）存储时加密

### 13、权限变更日志：记录角色 / 权限的变更历史

### 14、水印：页面添加动态水印（显示当前用户名）

### 15、异常拦截：接口请求失败时自动重试 / 友好提示
